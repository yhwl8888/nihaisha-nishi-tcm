## 精华要点
- **阴阳平衡与治未病**：确立“法于阴阳，和于术数”“恬惔虚无，真气从之”的养生观，揭示“阴平阳秘，精神乃治”的生命动态平衡机制，强调“不治已病治未病”的核心预防思想。
- **藏象学说与生命物质**：系统定义十二脏腑之职及奇恒、传化之腑的区别；阐明水谷精微化生气血的路径，并将人体生命物质细分为“决气”（精、气、津、液、血、脉）六种形态。
- **病机总纲与百病始生**：定调“邪气盛则实，精气夺则虚”；提出著名的“病机十九条”，奠定“谨守病机，各司其属”的辨证金标准；揭示外感虚邪贼风与内伤情志相合致病的传变规律。
- **营卫循行与经络系统**：明晰“清者为营，浊者为卫”，营行脉中、卫行脉外，一昼夜五十营的循行规律，揭示卫气失常导致的失眠与嗜睡病机；梳理十二经脉、五输穴及四海系统。
- **脉诊原理与望诊大局**：极度强调“胃气”对决断死生的决定作用（无胃气曰逆，真脏脉见皆死）；确立人迎寸口、尺肤诊法，结合明堂（鼻）五色面诊，构建了察色按脉的综合诊断体系。
- **体质分类与治病求本**：按阴阳五行将人分为“五态之人”与“二十五人”，强调诊病须结合患者社会地位与心理变化（如“尝贵后贱”名脱营，“尝富后贫”名失精），做到因人制宜。
- **治则治法与五味调脏**：确立“寒者热之”“逆者正治，从者反治”及“通因通用”等正反治法；提出顺应五脏苦欲的五味用药法则，并详述五味偏嗜及五劳所伤的危害。
- **针刺准则与九针体系**：明确九针形态与主治，总结九刺、十二刺等操作；强调针刺须顺应天时日月气血盛衰，明辨缪刺（左病取右）与巨刺，严守刺法禁忌。
- **五运六气与天人相应**：奠定五运六气推演基础，揭示岁候胜复、刚柔失守对气候与疫毒（五疫）爆发的影响，强调治病须顺时避忌、“先取化源”。

| **阴平阳秘** | 阴气平顺内守，阳气固密于外，两者互相协调制约，是人体健康的本源状态，离决则精气乃绝。 |
| **奇恒/传化之腑** | 奇恒之腑（脑髓骨脉胆女子胞）藏精气而不泻；传化之腑（胃肠三焦膀胱）传导水谷糟粕，泻而不藏。 |
| **病机十九条** | 《至真要大论》核心辨证体系，将风寒暑湿燥火及各类症状归系于五脏，为审察病机的金标准。 |
| **五态之人** | 依阴阳偏盛将人分为太阴、少阴、太阳、少阳、阴阳和平五类，反映不同的神志性格与气血特征。 |
| **十二原与五输穴** | 十二原为五脏受气之源；五输（井荥输经合）为经气游行出入之部位，四时取穴各有其规律。 |
| **营卫与五十营** | 清者为营，浊者为卫。一昼夜在体内循行五十周（五十营），卫气留于阳不得入阴则失眠，反之则多卧。 |
| **人迎寸口** | 人迎主阳经（外），寸口主阴经（内）。通过对比两者脉象大小倍数判断阴阳经盛衰及“关格”死证。 |
| **五实五虚** | 决死生的危重证候。五实（脉盛皮热等，邪极盛）；五虚（脉细皮寒等，正极衰）。 |
| **五运六气** | 结合天干地支推演岁候气候变化（木火土金水、风寒暑湿燥火）及对人体疾病与五疫流行影响的宏观学说。 |
| **脱营失精** | 情志内伤致病。“尝贵后贱”名脱营，“尝富后贫”名失精，强调诊病必问人事起居与精神状态。 |
| **开鬼门，洁净府** | 治疗水液代谢障碍（如水肿）的大法，即发汗解表（鬼门指汗孔）与利水渗湿（净府指膀胱）。 |
| **肠覃/石瘕** | 妇科腹部积块病机。肠覃客于肠外（月经如常），石瘕客于子门（闭经），皆可致腹大如怀子状。 |

## 风险与边界
- **高风险针刺与外科操作**：本资料包含大量古代针刺深浅、刺络放血（“尽出其血”）、特定危险穴位刺禁（如刺中眸子、刺中心）及铍针排脓等论述。现代临床需严格遵循无菌及解剖学禁忌，强刺激或放血操作极易引发大出血或神经损伤，非执业中医师严禁尝试。
- **急危重症提示**：文中提及的“癫狂”“尸厥”“关格”“霍乱”“脱痈”“猛疽”及五脏气绝等属古代急危重症，现代遇此类症状应立即寻求现代医学急救，绝不可仅盲信古法或延误治疗。
- **剧毒药物警示**：文内辟疫古方“小金丹”含有辰砂、雄黄、雌黄等重金属剧毒矿物药，具有极高毒性与致死风险，**绝对禁止自行配制或服用**。
- **妇科与孕产边界**：涉及子宫病变（石瘕）与验孕脉象（手少阴脉动甚），不可作为现代临床唯一判断标准。若有停经或腹部包块，需结合现代医学影像及生化检查确诊。

## 精华要点

- **生命周期与生长规律**：女子以七岁为生理周期（14岁天癸至，49岁任脉虚），男子以八岁为生理周期（16岁精气溢泄，64岁天癸竭）。顺应自然规律，节制欲望，是养生延年的根本。
- **五行与五脏对应**：肝属木（青、酸、筋、目），心属火（赤、苦、脉、舌），脾属土（黄、甘、肉、口），肺属金（白、辛、皮毛、鼻），肾属水（黑、咸、骨、耳）。五色五味直接关联脏腑气血。
- **阴阳辨证核心**：阴有形，阳无形。“阳化气，阴成形”，所有肿瘤块物（癥瘕）皆视为“阴实”（里寒累积、阳气不能化水）。治疗核心在于恢复阴阳平衡，固守阳气。
- **六经开合枢机制**：太阳与太阴主“开”（表），阳明与厥阴主“合”（里），少阳与少阴主“枢”（半表半里）。外邪侵入多循六经传变，掌握开合枢有助于截断病势。
- **脉象与胃气**：正常脉象必须“有胃气”（从容和缓）。“浮滑为阳，沉涩为阴”。若出现无胃气的“真脏脉”（如肝脉全弦如刀刃、心脉坚硬如薏仁），多为预后不良的死证。
- **疾病传变与五邪**：脏腑病变遵循生克制化。正邪（本脏自病）、虚邪（母病及子）、实邪（子病犯母）、微邪（相克之脏乘虚而入）、贼邪（被克之脏反侮）。治则为“实则泻其子，虚则补其母”。
- **水液代谢枢纽**：水液代谢依赖脾之运化、肺之通调、肾之气化。三焦为决渎之官，肾为胃之关。水肿病多为“阴实”，需通过发汗或利尿恢复气化。
- **情志与脏腑损伤**：心藏神，肺藏魄，肝藏魂，脾藏意，肾藏志。大怒伤肝，暴喜伤心，悲哀伤肺，思虑伤脾，惊恐伤肾。情志过极会直接导致气机逆乱（如怒则气逆，恐则气下）。
- **标本传变与治则**：治病需分清标本缓急。“中满者，先去中满”，即脾胃受困时先解决中焦堵塞；“小大不利而后生病者治其本”，先通调二便。急则治其标，缓则治其本。
- **四时刺法与禁忌**：针刺需顺应四时（春夏浅刺，秋冬深刺）。严禁误刺五脏核心部位（如深刺胸背伤及心肺），强调“刺家不诊，听病者言”与“神无营于众物”的专注度。

## 核心条目

| 术语 | 蒸馏说明 |
| :--- | :--- |
| **天癸** | 促进人体生殖机能发育成熟的物质。女子十四岁至，四十九岁竭；男子十六岁至，六十四岁竭。 |
| **真脏脉** | 五脏真气败露的脉象，无“胃气”（不柔和），如肝脉极弦如刀刃、肾脉坚硬如石。出现此脉多提示病情危重。 |
| **阴实** | 阴气极盛而阳气虚衰导致的实质性病变（如肿瘤、积水）。病机多为里寒累积、湿寒凝滞，阳气无法化气。 |
| **缪刺法** | 一种交叉刺法，遵循“左病右治，右病左治”。常用于病邪在络脉而非主经时，如在井穴或青筋处点刺。 |
| **五实五虚** | 决断生死的指征。五实（脉盛、皮热、腹胀、二便不通、闷瞀）若能出汗通便可生；五虚（脉细、皮寒、气少、泄利、饮食不入）若能进食可生。 |
| **腠理** | 泛指皮肤、肌肉的纹理及间隙，是卫气散布和外邪入侵的门户。 |
| **飧泄** | 指完谷不化的腹泻病症，多由脾胃虚寒、湿气下注引起。 |
| **风水** | 水肿病的一种。因出汗遇风，邪客玄府（毛孔），内应于肾与脾，表现为面浮浮肿、恶风、脉浮。 |
| **阴阳交** | 温病死证之一。热病汗出后复热，且脉躁疾、不能食、狂言。提示邪胜正衰，精气耗竭。 |
| **母子补泻法** | 针灸与用药核心法则：“实则泻其子，虚则补其母。子能令母实，母能令子虚”。 |

## 风险与边界

- **高危方剂警示**：笔记中提及的《伤寒论》方剂（如十枣汤、大承气汤）为峻烈之剂，适用于极度明确的“实证”或“水饮阻结”，原笔记中提及的剂量及“放手一搏”等个人经验已隐去，**严禁非专业人士自行尝试**。
- **刺血与针灸风险**：文中提及的“青筋放血”、“缪刺井穴”、“深刺至骨”、“刺缺盆/脑户等禁穴周边”等操作具有极高的感染、神经损伤或内脏穿孔风险（如气胸）。仅作古代针灸理论学习索引，不构成现代临床操作指南。
- **急重症与肿瘤提示**：笔记将肿瘤、尿毒症、急腹症（如盲肠炎/肠痈）、大出血（吐血/崩漏）归为“阴实”或急症。学习者需知此类疾病在现代医学属于高危急重症，必须立即就医，不可仅凭笔记自行判断或用药。
- **孕产妇禁忌**：提及的“女子胞”相关病变及孕期症状，严禁自行使用活血化瘀药物或针刺下腹部及特定经络，以免导致流产或大出血。
