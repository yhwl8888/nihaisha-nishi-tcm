# Nihaisha 中医资料索引

> 医疗边界：本索引只用于课程学习和资料路由；涉及真实症状、诊断、处方、剂量、针灸操作、急症、孕产儿童、肿瘤或附子等高风险内容时，必须保持非医疗建议边界。

本 skill 用于按课程模块渐进加载资料，避免加载全部转写。默认入口只保留课程蒸馏、课程笔记、截图证据、课程 PDF 校对证据和古籍方证索引；汉唐文章、事实评论、秘方手法、外部医案杂集、图片/可执行文件/课件资产等旧杂集不作为默认来源。资料覆盖 `01.针灸课程`、`03.黄帝内经课程`、`05.神农本草课程`、`07.伤寒论课程`、`09.金匮要略课程`、`11.仲景心法传讲`、`13.人纪之临床案例`、`14.人纪之八纲辨证`、`15.扶阳论坛`、`18.倪师易筋经`、`19.梁冬对话倪师`、`20.倪师斯坦福大学演讲` 与 `22.倪海厦天纪` 的课程蒸馏、逐课资料和截图证据索引，并接入 `references/pdf-evidence/` 作为公开安全的 PDF 页级溯源证据层。梁冬对话与斯坦福演讲为文字资料模块；其余已列截图证据的模块可按截图索引检索课程画面。

## 检索建议

- 症状问题：先 `symptom-index.md`，再 `six-channel.md`，最后 `formula-patterns.md`。
- 白话问题：先 `beginner-questions.md`，把普通话问题转成分水岭，再进入症状/六经/方证模块。
- 方剂问题：先 `formula-patterns.md`，必要时回到 `six-channel.md` 看病机层次。
- 逐课问题：用 `lesson-map.md`，再按课次查 `screenshot-evidence.md`。
- 金匮问题：先 `jingui.md`；需要板书/实操证据时查 `jingui-screenshot-evidence.md`。
- 仲景心法问题：先 `zhongjing-xinfa.md`；需要病机图、方药板书、眼诊、脏腑关系或重症课程观点证据时查 `zhongjing-xinfa-screenshot-evidence.md`。
- 临床案例问题：先 `clinical-cases.md`；需要病例板书、处方、病机图或重症案例证据时查 `clinical-cases-screenshot-evidence.md`。
- 八纲辨证问题：先 `bagang.md`；需要课程画面或字幕代表帧时查 `bagang-screenshot-evidence.md`。
- 扶阳论坛问题：先 `fuyang.md`；需要白板、PPT、案例幻灯片、阴阳/水火既济图或重症案例视频证据时查 `fuyang-screenshot-evidence.md`。
- 易筋经问题：先 `yijinjing.md`；需要动作示范、姿势、呼吸发音或功法画面证据时查 `yijinjing-screenshot-evidence.md`。
- 梁冬对话问题：先 `liangdong.md`；本模块暂时没有截图证据索引。
- 斯坦福演讲问题：先 `stanford.md`；本模块暂时没有截图证据索引。
- 天纪问题：先 `tianji.md`；需要易经、八卦、阳宅、风水、紫微斗数、命宫、四化、先天卦/后天卦、天干地支或占卜术数板书证据时查 `tianji-screenshot-evidence.md`。
- 黄帝内经问题：先 `huangdi.md`；需要板书、PPT、五行、四时、脉诊、藏象或病机图证据时查 `huangdi-screenshot-evidence.md`。
- 神农本草问题：先 `bencao.md`；需要药性、药物分类、性味归经、剂型剂量或配伍证据时查 `bencao-screenshot-evidence.md`。
- 针灸问题：先 `acupuncture.md`；需要穴位图、经络图、手法或实操证据时查 `acupuncture-screenshot-evidence.md`。
- 文字笔记问题：按课程先看对应视频课程引用，再打开 `notes-acupuncture-dacheng.md`、`notes-huangdi.md`、`notes-bencao.md`、`notes-shanghan.md` 或 `notes-jingui.md` 做补充。
- PDF/古籍溯源问题：先打开 `ebooks.md` 和 `pdf-evidence/index.md`，限定使用课程蒸馏、课程 PDF 证据和课程相关古籍方证索引；旧杂集不作为默认依据。
- 音频合集问题：打开 `audio-collection.md`，按目录映射回已蒸馏课程引用。
- PDF 溯源问题：打开 `pdf-evidence/index.md`；用 `python scripts/search_pdf_evidence.py <术语...>` 或 `rg` 检索 `pdf-evidence/evidence-cards.jsonl` / `pdf-evidence/term-index/<module>.json`，再用 `pdf-evidence/source-manifest.json` 解析 PDF 文档 ID，引用格式为 `pdf-evidence:<doc_id>#p<page>`。
- 截图问题：用 `python scripts/search_screenshots.py <自然语言问题或关键词...>`，脚本会归一化组合查询；关键词可以是方名、病机、六经名、课次或时间点。
- 结构设计问题：先看 `learning-entry.md`，优先按用户任务组织，而不是按课程目录组织。
- 医疗安全：真实症状只做课程学习分析，不直接给个人处方；高风险方药和急危重症必须提醒面诊或急诊。
