## 精华要点

- **太阳病与解表法则**：太阳病为表证，主提纲为“脉浮，头项强痛而恶寒”。辨证分中风（桂枝汤）、伤寒（麻黄汤）、温病。发汗法度重在“微似有汗”，忌大汗亡阳伤津；表里同病时，一般遵循“先解表后攻里”或表里双解。
- **少阳枢机与和解法**：病入半表半里，症见往来寒热、胸胁苦满、心烦喜呕。主方小柴胡汤，治法重在和解少阳，**严禁汗、吐、下**（误下易致惊悸）。
- **阳明实热与急下存阴**：阳明病核心病机为“胃家实”（燥屎）。经热用白虎汤类；腑实用三承气汤（大、小、调胃）。若见高热大汗、腹满痛拒按等津液枯竭危候，需遵循“急下存阴”原则以保胃肠津液。
- **太阴温运与少阴寒热**：太阴病属脾胃虚寒湿盛，重在温中健脾，忌攻下；少阴病（心肾）提纲为“脉微细，但欲寐”，有寒化（真武汤、附子汤温阳利水）与热化（黄连阿胶汤滋阴清火）之分。
- **厥阴错杂与生死危候**：厥阴病多寒热错杂（主方乌梅丸）。若见“除中”（胃气将绝之暴食）或“戴阳”（下焦极寒逼虚阳上浮面赤），皆属阴阳离决之危候；下利脉实为死候，下利微热脉缓为顺证。
- **水饮停聚辨治体系**：表水不解用五苓散（化气行水）；中焦停水用苓桂术甘汤（温化）；水停胸胁为悬饮重证用十枣汤（攻逐）；少阴阳虚水泛用真武汤（温阳定悸）；水热互结下焦用猪苓汤（滋阴清热利水）。
- **误治变证（结胸、痞、火逆）**：表证误下易致邪陷。邪与痰水实热结于胸为“结胸”（按之痛，属实，大陷胸汤）；结于心下为“痞”（满而不痛，属气滞，泻心汤类）。误用火攻强汗易致“火逆”（亡阳惊狂、奔豚）。
- **瘀血与黄疸辨治**：下焦蓄血见如狂、少腹急结，轻用桃核承气汤，重用抵当汤；湿热实并存之阳黄用茵陈蒿汤，无里实之身黄用栀子柏皮汤，兼表证发黄用麻黄连翘赤小豆汤。
- **极寒极虚之阴实**：脏腑寒湿相结之“脏结”、骨三阴（囟骨、耳后高骨、膝盖骨）现肿大等，多提示病邪深陷阴实，属危重无阳之证，切忌误用寒凉攻下。
- **脉证合逆原则**：中医极重脉证相应。症虚脉大，或症实脉小，皆属脉证不合之危象；结脉、代脉属阴（伤阴血），促脉属阳。

## 核心条目

| 术语 | 蒸馏说明 |
| :--- | :--- |
| **胃家实** | 阳明病核心病机，指胃肠（大肠、小肠、胃）燥热实结，表现为身热、汗自出、不恶寒反恶热及便秘（燥屎）。 |
| **结胸与心下痞** | 皆因误下致邪陷。结胸为邪与水/热结于胸胁，特征为“按之痛”（大陷胸汤）；心下痞为中焦气机痞塞，特征为“按之濡/满而不痛”（泻心汤类）。 |
| **脾约** | 脾虚津液不布，导致小肠干燥，大便硬结如羊屎粒。治宜和缓润下（麻子仁丸）。 |
| **火逆** | 误用火法（火劫、烧针、误灸）强行逼汗，导致伤津、亡阳、惊狂、谵语或气上冲（奔豚）的变证。 |
| **脏结与阴实** | 阴寒凝聚脏腑，无阳证表现（如饮食如故、时时下利、舌苔白滑）。属难治之阴证危候，忌用攻下。 |
| **热入血室** | 妇女经期感受外邪，邪热乘虚侵入子宫，致寒热往来如疟状，或夜间谵语。治以和解（小柴胡汤）或刺期门穴。 |
| **除中** | 厥阴病死候。本不能食，在无微热与阳脉恢复的情况下突然暴食，为胃气将绝、阴阳离散的回光返照。 |
| **戴阳** | 阴寒极盛于下，逼迫虚阳上浮于上（如面色微赤），伴下利清谷、脉微欲绝。需急用通脉四逆汤等破阴回阳。 |
| **桂枝汤 / 麻黄汤** | 太阳病基础方。桂枝汤治中风（调和营卫，解肌发表）；麻黄汤治伤寒（发汗解表，宣肺平喘）。 |
| **三承气汤** | 治阳明腑实。大承气汤攻大肠（无矢气、潮热）；小承气汤和胃气（有矢气、腹微满）；调胃承气汤缓下热结。 |
| **泻心汤系列** | 治心下痞。生姜泻心汤治水饮；甘草泻心汤治脾胃虚寒；大黄黄连泻心汤治胃中虚热（麻沸汤泡服取其气）。 |
| **乌梅丸** | 厥阴病主方。治寒热错杂之“吐蛔”及久利。具酸收、辛散、苦降之性，能安蛔温脏。 |
