# 方剂方证模块

> 医疗边界：本文件仅作课程学习、资料检索和中医理论整理；涉及真实症状、诊断、处方、剂量、针灸操作、急症、孕产儿童、肿瘤或附子等高风险内容时，不能作为个人医疗建议或自行操作依据，应咨询合格医疗专业人员。

> 用于按方名、六经、症状群检索倪海厦《伤寒论》课程里的方证。仅作学习整理，不输出个人处方剂量。

## 使用方法

- 方名明确：直接检索方名，给“方证、鉴别、禁忌、课程课次、截图证据”。
- 方名不明确：先从 `symptom-index.md` 和 `six-channel.md` 定位候选方证。
- 涉及承气汤、四逆汤辈、附子、抵当汤、大陷胸汤、癌症/阴实时，必须提示高风险。

## 高风险方剂提示词

遇到以下方名或病机，回答必须加重安全提示：

- 承气汤类：大承气汤、小承气汤、调胃承气汤，尤其“急下存阴”“热结旁流”。
- 附子类：四逆汤、通脉四逆汤、白通汤、白通加猪胆汁汤、真武汤、麻黄附子细辛汤。
- 峻下/破血/逐水：抵当汤、大陷胸汤、桃核承气汤。
- 癌症、肿瘤、阴实、附骨脉、主动脉剥离等重症相关内容。

标准安全句：

```markdown
这部分是倪海厦课程里的经方理论学习，涉及峻烈攻下/破血/附子回阳/危重症判断，不能作为个人用药建议。真实身体情况必须由执业中医师面诊辨证；若有急性胸腹痛、神志异常、脱水、休克样表现，应立即急诊。
```
