## 人纪-针灸篇.pdf

以下内容汇总《针灸大成》（人纪-针灸篇）四部分笔记的核心要点，便于中医理论学习与文献检索。

## 精华要点

- **针灸时机与解剖原则**：气血大虚、大汗、酒醉时不宜下针。进针需在“肌肉与骨、肌肉与筋或两肌之间的缝隙”，绝不可直接扎在骨、筋或血管上。
- **特定时辰与流注诊断**：结合“十二经纳地支”（如寅时肺、丑时肝、子时胆），若特定时辰规律发病（如半夜定时苏醒提示脏病），可辅助定位病变经络。
- **五输穴与子母补泻**：阴经五输穴（井荣俞经合）属性为“木火土金水”，阳经为“金水木火土”。循“虚则补其母，实则泻其子”原则；遇“泻井当泻荣，补井当补合”之特殊情况需灵活变通。
- **经典配穴治疗法**：
  - **原络治疗**：病经取“原穴”，表里经取“络穴”。
  - **俞募治疗**：治内脏病，背部“俞穴”（如肺俞）与胸腹“募穴”（如中府）同用。
  - **会郄治疗**：八大会穴（如骨会大杼、血会膈俞）配合郄穴（主消炎镇痛）合用。
- **脏腑气化与经乳同源**：小肠（火）加热下焦，使膀胱（水）气化升腾。妇科理论认为“奶水等于月经”，受心阳向下消导至子宫；若心力弱导致停滞，在乳房易结乳岩，在子宫易生痞块。
- **心与心包界定**：心脏为“君主之官”本身不受病，病邪皆在心包络。心脏移热于小肠，若手足冰冷、失眠，常提示心阳不足、小肠火衰，为心脏病发之前兆。
- **寒热虚实辨证**：隐痛喜按、分泌物清白多属虚寒，宜用灸法（中脘、足三里）；拒按剧痛、分泌物深黄粘稠多属实热，宜用针刺。左侧少腹压痛多提示瘀血（如肌肤甲错）。
- **气机导引与对称治疗**：上焦大穴（如肩井）行针引气向下时，必须配伍下焦穴位（如足三里）以防气血逆乱晕厥。痛症可采用左病右治，或类比取穴（如承山类比肛门治腰痛）。
- **婴幼儿针刺原则**：十四岁以下幼儿采用“单刺”不留针（针入捻转即出）或采用指针推拿（如开天河水）；十四岁以上视同成人留针并施以补泻。

| **十三鬼穴** | 孙思邈所创特定穴位序列，主治癫、痫、狂等因痰涎壅塞引起的神志失常，有严格的下针顺序与男女左右捻转规定。 |
| **奇经交会八穴** | 公孙/内关（冲脉/阴维主胃心胸）、后溪/申脉（督脉/阳跷主颈项督脉）、临泣/外关（带脉/阳维主侧面）、列缺/照海（任脉/阴跷主肺系咽喉）。 |

## 风险与边界
- **急救与强刺激警示**：资料提及的十宣/百会/井穴放血、回阳九针、十三鬼穴、癫痫/心脏骤停急救、化脓灸等，均为古代重症处置理论，**仅作课程资料索引，严禁非执业医师模仿操作。**
- **孕产妇禁忌**：合谷、三阴交、昆仑、至阴等穴位涉及堕胎/安胎及下胎盘机理，补泻手法极具风险；腹部与骶部穴位（八髎穴等）在孕期通常禁针禁灸，误操作易致落胎。
- **解剖危险区域**：缺盆、巨骨、肩井深刺易致气胸；颈部（天突、人迎）、面部（承泣）、腋下深部有大血管或脏器，严禁深刺及大幅度提插捻转。内脏出血（如胃出血）时禁用本经本穴（如足三里）以免加重出血。
- **肿瘤与重症边界**：关于乳岩（乳腺癌）、血癌、肝癌、脑瘤的针灸消导理论（如火针/铍针刺痞块、毒药外敷灸），仅供中医病机探究与文献学习，**绝不构成替代现代规范化疗/手术的医疗建议。确有疾病请务必遵从正规医疗机构施治。**

## 精华要点

- **辨证与取穴原则**：遵循“左病右治、右病左治”的对称治疗，以及“病进（向身体蔓延）、病出（向四肢退）”的传变规律。取穴常结合同身寸，并根据病邪位置决定针刺深浅（如冬天宜深，腹深似井、背薄似饼）。
- **经络气血流注**：十二经脉配合地支有明确的时辰规律（如寅时肺、卯时大肠、辰时胃等），临床常据此进行时辰辨证与子母补泻（如肝病在凌晨1-3点发作或失眠）。
- **痛症辨证**：痛症按压时，若“拒按、痛处固定”多为实证、瘀血，宜用泻法或放血；若“喜按”多为虚证，宜用补法。
- **五俞穴应用**：各经均有“井、荣、俞、经、合”五穴。治疗遵循“虚则补其母，实则泻其子”；急救多取井穴，内脏病多取合穴；原络治疗法常用于表里经同病。
- **俞募治疗法**：主要用于内科脏腑疾病，选取背部的俞穴（属阳）与胸腹部的募穴（属阴）相配合，俞穴不论虚实寒热皆可使用。
- **针刺补泻**：补泻手法涉及呼吸法、迎随法、提插法与捻转法。阳数为补（如奇数、九数），阴数为泻（如偶数、六数）。气至（得气）表现为酸麻胀或针孔周围呈粉红色。
- **艾灸法则**：适用于寒证、虚证及部分急症（如毒虫咬伤）。隔姜灸温通，隔盐灸（神阙）回阳救逆，隔蒜灸多用于久咳或毒疮。热证、多汗、面部及孕妇禁灸或慎灸。
- **奇经八脉与八脉交会**：任督二脉统领阴阳，冲脉为血海。临床常配合“灵龟八法”按日干支开穴，或使用八脉交会穴（如公孙配内关、列缺配照海）治疗全身性复杂疾病。

## 核心条目

| 术语 | 蒸馏说明 |
| :--- | :--- |
| **十二经纳地支** | 经气流注时间：肺寅（3-5点）、大卯（5-7点）、胃辰（7-9点）、脾巳（9-11点）、心午（11-13点）、小未（13-15点）、膀胱申（15-17点）、肾酉（17-19点）、心包戌（19-21点）、三焦亥（21-23点）、胆子（23-1点）、肝丑（1-3点）。 |
| **子母补泻** | 根据五行生克及五俞穴属性进行治疗的法则。“虚则补其母，实则泻其子”，如胆经（木）实证，可泻其子穴阳辅（火）或用同属相生经络的穴位。 |
| **俞募治疗法** | 脏腑病证的核心疗法。取背部膀胱经的脏腑“俞穴”搭配胸腹部对应的“募穴”（如胃病取中脘配胃俞，肺病取中府配肺俞）。 |
| **原络治疗法** | 主客原络法。主病经取“原穴”，相表里的客经取“络穴”。用于两经并发疾病或表里同病（如肺为主、大肠为客，取太渊配偏历）。 |
| **烧山火 / 透天凉** | 经典强刺激补泻手法。**烧山火**为强补法，分天、人、地三部重插轻提（九数），治虚寒厥冷；**透天凉**为强泻法，重提轻插（六数），治高热实热。 |
| **导痰术** | 专治顽痰、癫痫的针刺手法。常取胃经络穴“丰隆”，进针得气后退至皮下，改变角度斜刺并捻转引气，留针以化痰。 |
| **四花灸** | 特定灸法，常取背部膈俞、胆俞（或类似组合），用于治疗久病虚损、气血大亏、肺部重疾（如肺痨、严重咳喘）。 |
| **三皇穴** | 脾经的阴陵泉（天皇）、地机（地皇）、三阴交（人皇）。为治疗水肿、泌尿生殖系统及妇科病（如下焦虚寒、痛经）的核心要穴组合。 |
| **灵龟八法** | 结合天干地支此时辰的“八脉交会穴”开穴法。强调“先开主穴，后开配穴”（如临泣配外关），用于疑难杂症的应急或起手治疗。 |
| **五脏气乱** | 因高处跌落或严重外伤导致脏腑气机逆乱，临床常取天柱、大杼等穴位平抑气机，不补不泻。 |
| **痞块 / 积聚** | 腹部或体表的硬块（良性或恶性）。诊断常参考“痞根穴”（第13椎下旁开3.5寸）。治疗强调“行阳化阴”，多用艾灸或特定消导穴位。 |

## 风险与边界

- **急症与重症警示**：笔记中提及的中风急救（十宣/人中/百会放血）、溺水急救、癫痫发作、心脏急救（极泉/少府/关元）、大出血、尸厥等，仅作为中医针灸理论的病机与治法索引。现实中遇到此类急症，必须立即寻求现代急诊医疗救治。
- **高风险针刺手法**：烧山火、透天凉、报刺、深刺透穴（如阴陵泉透阳陵泉、条口透承山）、眼区针刺（睛明、球后）及强刺激的“导痰术”，操作不当极易造成气胸、神经损伤、脏器穿孔或晕针，非专业执业医师严禁操作。
- **放血与火罐风险**：三棱针放血（如委中、耳尖、十宣）及火罐吸脓血属于创伤性操作，有严重的感染风险，严禁自行模仿。
- **孕产与儿童禁忌**：孕妇严格禁针合谷、三阴交、肩井及腹部穴位（可能导致流产）。14岁以下儿童针灸需采用“单刺不留针”法。婴幼儿推拿退热手法仅供理论参考。
- **肿瘤与外用秘方**：提及的肺癌、乳癌、脑瘤、脾肿大、肝硬化等重疾的针灸思路仅供中医学习探讨。文中提及的“巴豆、硫磺、生附子、麝香”等属于剧毒或峻猛中药，相关外敷灸法（如伤寒结胸灸、化脓灸）具有极高危险性，可能导致严重皮肤坏死或中毒，绝对禁止自行配制和使用。
